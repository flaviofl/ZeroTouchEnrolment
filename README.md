# Zero Touch Enrolment
Android Zero Touch Enrolment

## Prerequisites
- A Google Zero Touch enrolment account (set up via your device reseller or directly with Google)
- Supported MDM
- Zero Touch compatible devices (Android 9.0+ from supported manufacturers)
- Zero Touch Enrolment devicea reseller
